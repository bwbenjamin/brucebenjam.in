Senior Project

'Prep' describes the working title for an app that I designed and prototyped for my senior project. During the spring of 2021, I completed a front-end demonstration of the application within Figma along with documentation to explain the ideas behind the application and track my progress.

GrC 357 Shirt

A screen-printed t-shirt design I created for my Specialty Printing Technologies course. Much like the mug I designed, this shirt demonstrates fundamental understanding of traditional screen printing, and features a design of my own creation, using Illustrator.

GrC 357 Mug

A mug I created for my Specialty Printing Technologies course. This mug demonstrates a comprehensive understanding of short run Dye Sublimation and showcases a fun design I created using Illustrator!

GrC 316 Stickers

A small mass of stickers that I designed with Illustrator and printed for my Flexographic Printing Technology course. These CMYK + UV stickers were a great introduction into the mechanics of the Flexo process and presented themselves as an opportunity to design small scale graphics.

GrC 329 Package Prototye

This is a simple render done in Adobe Dimension to showcase the packaging for a product that I created with a group for my printed electronics class. Our group created a prototype for banana bread package that would mix itself with a servo motor and beep when finished. Designing this prototype was primarily an exercise in working with simple electronics and building code.

GrC 338: Web Graphics

The graphics I created for a website as part of my Web Design course. Though the website was initially a demonstration of how knowledge of website construction translated into a website builder, I took the opportunity to create a couple rendered mockups for a coffee storefront in Adobe Dimension and Illustrator.

GrC 337 Perfume Package

A packaging design project as part of my consumer packing class. This project was eventually physically prototyped and submitted to a PIASC design competition.

GrC 318 Tech Paper

This project was an exercise that involved redesigning the text for an article written for Aiga design magazine using Adobe InDesign. This project allowed me to better understand the mechanics of InDesign and the principles of text flow.
